# Directory Submission Kit

Ready-to-paste copy and step-by-step instructions for getting **Draft** listed
across awesome lists, Product Hunt, and AI-tool roundups.

Everything in this folder is a **prepared submission** — the actual submit step
(opening a PR on a third-party repo, posting on Product Hunt, filling a tool
directory form) requires a logged-in human and can't be automated from this repo.
Copy the relevant block, follow the steps, then tick the tracker below.
